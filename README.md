# Java Strings Practice 🚀

Welcome to my Java Strings repository! This is part of my journey to master Data Structures and Algorithms (DSA) in Java. Here, I practice various String manipulation techniques and concepts.

## 📚 Topics Covered
- **String Basics**: Declaration and Input methods (`next()`, `nextLine()`).
- **Concatenation**: Joining strings using `+` operator.
- **Length Function**: Understanding `str.length()`.
- **charAt() Function**: Accessing individual characters at specific indices.
- **CompareTo Function**: String comparison logic (Lexicographical).
- **SubString Function**: Extracting parts of a string.
